# keylogger
Description
This project is a Python educational experiment that demonstrates how keyboard events can be captured and written to a local log file.
It is intended strictly for learning purposes, such as understanding event listeners, file handling, and basic automation concepts.
Purpose
Learn how keyboard event listeners work in Python
Practice file input and output
Understand the basics of logging systems
Explore security concepts in a controlled environment
Important Notice
This project is created only for educational and ethical use.
Do not use this code on devices or accounts you do not own.
Do not collect any data without explicit user consent.
Misuse of keylogging software may be illegal.
